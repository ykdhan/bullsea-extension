# Bullsea Chrome Extension

"작은 불씨에서 시작된다." - 코인 투자 필수 tool, 불씨!

| 현재 지원되는 거래소 | 업데이트 날짜 |
| -------------------- | ------------- |
| `업비트`, `빗썸`     | 22.12.14      |

---

#### 로드맵

-   국내 5대 거래소 지원.
-   종목 별 실시간 접속 유저 현황.
-   실시간 채팅.
-   웹사이트 출시.
-   모바일 앱 출시.
-   유저 데이터 분석 > 고급 데이터.
